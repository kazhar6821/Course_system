# Coursera Courses Data Analysis

This repository contains a dataset and analysis scripts for Coursera courses collected from Coursera's catalog. The project focuses on exploring course attributes like difficulty level, ratings, skills taught, and course descriptions.

## Dataset

- The dataset includes 3,522 Coursera courses.
- Key columns:
  - **Course Name**
  - **Difficulty Level**
  - **Course URL**
  - **Course Description**
  - **Skills**

## Features

- Data cleaning and preprocessing of text data.
- Extraction and combination of keywords into a searchable `tags` column.
- Analysis of course difficulty distribution and ratings.
- Basic visualizations using matplotlib and seaborn.
